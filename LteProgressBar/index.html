    # LteProgressBar

    • Cores
    • Tamanhos
    • Percentual
    • Striped
    • Animação
    • Vertical

    O componente LteProgressBar fornece uma barra de progresso.

          <lte-progress-bar :percent="12" color="success"/>
    12%
    0.00%
    0.00%

    # Cores

    A barra de progresso tem 4 cores: success, primary, warning e danger. A cor é indicada pela propriedade color, do tipo String.

    12%
    12%
    12%
    12%

    # Tamanhos

    A barra de progresso tem 3 tamanhos: xxs, xs e sm. O tamanho é indicado pela propriedade size, do tipo String.

    12%
    12%
    12%

    # Percentual

    Para indicar o percentual preenchido da barra colorida, usa-se a propriedade percent, do tipo Number.

    10%
    50%
    80%

    # Striped

    Para deixar a barra de progresso listrada, usa-se a propriedade striped, do tipo Boolean, como true.

    50%

    # Animação

    Para deixar a barra de progresso animado, usa-se a propriedade active, do tipo Boolean, como true.

    50%

    # Vertical

    A barra de progresso também pode ficar na vertical, utilizando-se a propriedade vertical, do tipo Boolean, como true.

    50%
