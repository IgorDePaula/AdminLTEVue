    # LteActionButton

    • Tipos
    • Itens
    • Flat
    O LteActionButton representa um botão com um menu dropdown. São 2 o número de parâmetros obrigatórios: title, String, que representa o label do componente, e itens, Array, que representa os itens que serão renderizados no menu dropdown. Tem ainda o parâmetro type que indica a coloração do componente. Parâmetro este herdado do componente LteButton.

    <lte-action-button type="warning" title="options" :itens="[{}]"/>

    # Itens

    Cada item do array da propriedade itens precisa ter exatamente 2 propriedades: title que é a label da opção, e handler que uma function que será executada quando esta opção for clicada.

    [
  {
    title: 'Option 1',
    handler: () => console.log('log from option 1 handler')
  }
]

    # Flat

    Também tem a opção flat que o faz ter cantos de ângulo reto.

    <lte-action-button type="warning" :flat="true" title="options" :itens="[{}]"/>
